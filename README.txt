В папке data каждый txt файл представляет собой набор вопросов и вариантов ответ. Построение теста на данных вопросах основываются 
на следующих правилах:
1. Каждый вопрос отделяются от следующего пустой строкой
2. Каждый файл, кроме соглашения и вопросов к видам пищи, начинается с двух пустых строк
3. Первая строка всегда является вопросом, а все последующие строки вариантами ответов к нему. Одна строка - один вопрос или один
 вариант ответа, перенос строк здесь не допустим
4. Цифры в начале строк являются кодами, обозначающими одно из следующих условий:
	1 - Вопрос, превращающийся в открытый при выборе первого варианта ответа (Если да, то расскажите подробнее)
	2 - Вопрос, запоминающий несколько вариантов ответа
	3 - Если да, то следующий вопрос активен, иначе пропускается с сохранением в слдующий вариант ответа "нет"
	4 - с этой цифры начинается вопрос, в котором польлзователь выбирает количество приёмов пищи в день
	5 - С этого вопроса начинаются дополнительные приёмы пищи
	6 - С этого вопроса начинаются уточнения по потреблению каждого вида пищи (говядина и т.д.)

Также в папке data в файле config.py необходимо указать почту, с доступом к таблице, api бота и user_id администратора
